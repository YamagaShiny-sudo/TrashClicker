# TrashClicker
Une collection de pages avec un jeu incrémental, avec une page présentant les animaux débloqués.

Pour accéder à la page, voici le lien : https://yamagashiny-sudo.github.io/TrashClicker/
Si vous voulez accéder au jeu sans passer par le lien, en utilisant Godot, le fichier se trouve dans le dossier 'trashclicker'.
La page web du jeu ne fonctionne que si elle est sur serveur.
